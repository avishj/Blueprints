# Astro TS React

## What is this stack for?

This stack scaffolds modern frontend applications built with Bun, Astro, React islands, and strict TypeScript.

It is opinionated and maximalist by default, with motion (GSAP) and WebGL/shader (Three.js) foundations baked into the baseline.

## Usage

This stack is a Copier template.

See [SETUP.md](SETUP.md) for prerequisites and scaffold commands.
